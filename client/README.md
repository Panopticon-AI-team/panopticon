## [Installation](#installation)

### [Clone Repository](#clone-repository)

```bash
git clone git@github.com:Panopticon-AI-team/panopticon.git
```

```bash
cd panopticon/client
```

```bash
npm install
```

---

### [Start Development Server](#start-development-server)

```bash
npm run start
```
